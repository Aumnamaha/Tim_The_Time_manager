Install in 5 steps

You install it yourself.
That's the whole point.

No mystery installer. No account. Download the source code directly from GitHub, inspect every file if you want, and add it to your browser yourself. Total transparency.

Download ZIP from GitHub

Free · Open source · No account needed

1

Download the ZIP file

Click the button above to download Tim_the_time_Extension.zip from GitHub Releases. This is the complete source code — every file is open for you to inspect.

📁 Tim_the_time_Extension.zip — fully open source
2

Extract the ZIP to a local folder

Right-click the downloaded file and choose "Extract All" (Windows) or double-click (Mac). Place it somewhere permanent on your computer.

Right-click → Extract All → Choose folder → Extract
3

Open the Extensions page in your browser

Type the address in your browser's address bar and press Enter:

Chrome & Brave chrome://extensions
Microsoft Edge edge://extensions
4

Turn on Developer Mode

In the top-right corner of the Extensions page, flip the "Developer mode" toggle ON. This lets your browser load extensions from local folders.

💡 Why Developer Mode? Tim is open-source and not yet on the Chrome Web Store. This toggle simply allows local folder installs — it does not reduce your browser's security.
5

Click "Load unpacked" and select the folder

A "Load unpacked" button appears in the top-left. Click it, then select the Tim_the_time_Extension folder you extracted in step 2.

Load unpacked → Select Tim_the_time_Extension → Open
✓

Tim is active! 🎉

The Tim clock icon appears in your toolbar. Browse YouTube, Instagram or Facebook — Tim starts tracking immediately. First wellness reminder in 40 minutes!
